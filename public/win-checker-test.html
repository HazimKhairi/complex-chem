Available Functions

  • WinChecker.checkWinCondition() - Check for winners and update modal
  • WinChecker.getPiecesInHome(playerId) - Count pieces in home for a player
  • WinChecker.calculateScore(playerId) - Calculate player's score from ligands
  • WinChecker.declareWinner(playerId, score) - Manually declare a winner
  • WinChecker.hasPlayerWon(playerId) - Check if player has all pieces home
  • WinChecker.getGameStatus() - Get complete game status
  • WinChecker.reset() - Reset win checker state
